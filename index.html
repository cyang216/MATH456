Description

Advanced topics in applied statistics including multiple regression, multivariate methods, nonparametrics, analysis of covariance, bootstrap methods and others as appropriate. The statistical programming language R will be used.

Appropriate for biology, agriculture, nutrition, business, psychology, social science and other maojrs. 3 hours discussion.

This course will emphasize reproducible research. Students will work to create data products with a professional appearance, and well-documented code that is transparent and reproducible by others with minimal effort.
